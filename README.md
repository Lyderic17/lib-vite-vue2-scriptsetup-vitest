# Library

## Technos

- Vite
- Vue2
- Script setup (vite plugin)
- SASS
- Vitest
- Vue test utils
- Docker

## Commands


#### Pull image and mount container
```bash
    yarn up
```
#### Access container's shell
```bash
    yarn shell
```
#### Kill container
```bash
    yarn down
```

#### Launch unit tests
```bash
    yarn test
```

#### Build lib
```bash
    yarn build
```
